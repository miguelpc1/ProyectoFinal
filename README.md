# 📊 Insurance Company Analytics - Proyecto Final
## Descripción del proyecto
Este proyecto es una aplicación interactiva que se ha desarrollado en Python utilizando Streamlit para el análisis exploratorio de datos de una empresa aseguradora.
Uno de sus objetivos es analizar a los clientes en el proceso de renovaciòn de su póliza, el cual se ve influenciado por diferentes variables como el ingreso mensual, el canal por el cual fue contactado, entre otras.
Esta aplicación permite carga archivos con extensión .csv para realizar su respectivo análisis.
